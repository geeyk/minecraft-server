# minecraft-server

Servidor do Minecraft dedicado de Geeyk

## Instruções de instalação e configuração

Requisitos:

- Prism Launcher (última versão)
- Modpack
- URL do Servidor

### Prism Launcher

Baixe o **Prism Launcher** pelo [link oficial](https://prismlauncher.org/download/) aqui.
A instalação depende da sua plataforma.

Compatível com:

- Windows
- Linux
- MacOS
- Steam Deck.

### Instalação do Modpack

Abra o **Prism Launcher**

Adicione nova instância (`Add Instance`)

Selecione a opção Importar (`Import`)

**Copie** o Link abaixo e **cole** no caminho de download da janela de importação:

```text
link.com.brrsss
```

Clique em `OK` e aguarde a operação.
